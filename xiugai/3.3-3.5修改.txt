### 修改说明
原标题"无监督训练策略"不变，但需在开头和关键位置补充说明：训练阶段仅训练GATNet，WMMSE精调在推理阶段使用、不需要额外训练。

---

### 3.3.1 损失函数设计（修改后完整内容）

GATNet的训练采用无监督端到端方式，以网络加权和速率为优化目标，无需依赖WMMSE等传统迭代算法生成标注数据。由于速率R_i关于功率p_i可微，可直接通过梯度下降最大化总和速率。损失函数定义为负加权和速率：

L(θ) = -1/B · Σ_{b=1}^{B} Σ_{i=1}^{N} w_i · log₂(1 + γ_i^(b))

式中，θ为网络可学习参数，B为批次大小，上标(b)表示第b个训练样本。

为提升公平性与QoS保障，引入辅助正则项：

L_total = L + λ₁·L_fair + λ₂·L_qos

其中，公平性正则项采用Jain公平指数惩罚：

L_fair = -(Σ_i R_i)² / (N · Σ_i R_i²)

QoS违反惩罚项采用L2平方惩罚以重点抑制严重违反：

L_qos = 1/N · Σ_i ReLU(γ_th - γ_i)²

式中，λ₁、λ₂为正则化系数，分别取0.05和0.15。

需要指出的是，上述损失函数仅用于GATNet的训练阶段。在推理阶段，GATNet输出初始功率p^(0)后，WMMSE精调步骤依据其自身的优化准则——最小化加权均方误差——进行迭代，不依赖于GATNet的损失函数。因此，GATNet的训练目标是学会输出高质量初始解，使得WMMSE从该初始解出发能够以极少迭代次数收敛到接近全局最优的性能。训练损失中的QoS和公平性正则项对WMMSE精调阶段同样有益：良好的初始解意味着WMMSE无需在约束违反区域浪费迭代步骤，可以更快地进入可行域并聚焦于目标函数优化。

---

### 3.3.2 训练流程（修改后完整内容）

训练过程如算法3.1所示。采用Adam优化器，初始学习率10^{-3}，每500轮衰减为原来的0.9倍。训练数据通过随机网络拓扑生成器实时采样（每轮16个随机拓扑样本），确保模型在整个训练周期内接触约48,000个不同的网络实例，从根本上防止对特定拓扑配置的过拟合。

**算法3.1 GATNet训练算法**

输入：训练轮数T，批次大小B，学习率η，网络层数L，注意力头数K
输出：训练好的GATNet模型参数θ

1.  随机初始化网络参数θ
2.  for t = 1, 2, ..., T do
3.      生成B个随机网络样本 {G^(b)}_(b=1)^B
4.      for each sample G^(b) do
5.          计算节点特征X^(b)和边集E^(b)
6.          前向传播：p^(b) = GATNet(X^(b), E^(b); θ)
7.          计算SINR γ_i^(b)和速率R_i^(b)
8.      end for
9.      计算总损失L_total(θ)
10.     反向传播计算梯度 ∇_θ L_total
11.     更新参数：θ ← θ - η · Adam(∇_θ L_total)
12.     if t mod 500 == 0 then  η ← 0.9·η
13. end for;  return θ

训练完成后的推理流程如下：给定测试网络拓扑，首先将信道状态信息转化为干扰图数据，输入训练好的GATNet进行前向传播以获取初始功率分配p^(0)（约5 ms），随后将p^(0)作为WMMSE迭代的初始值，经8次迭代完成精细优化得到最终功率分配p*（约3 ms）。WMMSE精调阶段不涉及任何可学习参数，因此不需要训练，不增加模型存储和部署的复杂度。

值得注意的是，GATNet的训练过程完全独立于WMMSE——模型并不"知道"自己的输出将在推理阶段被WMMSE进一步优化。这种解耦设计的优势在于：若后续替换或升级WMMSE精调策略（如改用其他迭代优化算法），GATNet无需重新训练即可适配。

### 3.4 基线算法与对比框架（修改后完整内容）

为全面评估所提方案（GATNet及GATNet+WMMSE混合方法）的性能，选取以下4种代表性算法与GATNet系列方法构成完整的对比框架：

1. **WMMSE（加权最小均方误差）**：经典迭代凸优化方法，通过辅助变量变换将非凸和速率最大化问题分解为凸子问题串行求解。设置最大迭代次数为60次，收敛精度为10^{-4}。在本对比框架中，WMMSE同时承担两个角色：一是作为纯GATNet的性能参照（数据驱动 vs. 模型驱动），二是作为GATNet+WMMSE混合方案中第二阶段精调算法的核心组件。

2. **GA（遗传算法）**：启发式全局搜索方法，种群规模100，迭代代数200，精英保留数10，变异概率0.1。GA代表了不依赖梯度信息、通过随机搜索探索解空间的方法类别，用于验证GATNet的梯度驱动学习是否优于无梯度的启发式搜索。

3. **Equal Power（等功率分配）**：所有链路均分最大功率p_i = p_max/2，代表最简单的非智能基线，用于衡量智能功率分配方法相对于无优化的性能提升幅度。

4. **Random Power（随机功率分配）**：在[0, p_max]内均匀随机生成功率向量，重复100次取平均性能，作为方法性能的下界参考。

5. **GATNet+WMMSE（本文提出的混合方案）**：第一阶段以训练好的GATNet进行快速前向推理获得初始解，第二阶段以该初始解启动WMMSE迭代（仅8次迭代），实现数据驱动方法与模型驱动方法的优势互补。该方案是本对比框架中的核心评估对象，用于验证"端到端学习提供初值 + 经典优化局部精调"这一混合策略的有效性。

表3.1给出了各算法的计算复杂度与核心特性对比。

表3.1 各算法计算复杂度与特性对比

| 算法 | 计算复杂度 | 优化方式 | 是否需GPU | 推理延迟(ms) |
|---|---|---|---|---|
| WMMSE | O(N³·T_iter) | 迭代凸优化 | 否 | ~7.33 |
| GA | O(G·P·N²) | 启发式搜索 | 否 | ~63.62 |
| Equal Power | O(N) | 无优化 | 否 | ~0.01 |
| Random Power | O(N) | 随机采样 | 否 | ~0.01 |
| GATNet | O(K·|E|·D²·L) | 端到端学习 | 是 | ~7.03 |
| **GATNet+WMMSE** | **O(K·|E|·D²·L + N³·8)** | **学习+优化混合** | **是** | **~8.31** |

式中，T_iter为WMMSE迭代次数（全迭代60次），G和P为遗传算法的代数与种群规模，|E|为干扰图中边数，K为注意力头数，D为每头维度，L为GAT层数。GATNet+WMMSE中WMMSE迭代次数仅8次（为全迭代的13.3%），混合方案的复杂度主要由GATNet前向传播主导。

### 3.5 本章小结（修改后完整内容）

本章系统设计了GATNet模型架构及其与WMMSE的两阶段混合功率分配方案，主要贡献包括：

1. **边特征增强的多头图注意力机制**：通过将双向干扰信道增益作为边特征显式注入GATConv的注意力计算，使模型能够自适应区分不同干扰源的影响强度，契合无线干扰环境的非均匀特性。与标准GAT仅依赖节点特征计算注意力不同，GATNet的边特征设计为模型提供了直接的干扰强度信息。

2. **双跳跃连接与残差-层归一化架构**：层间残差连接（h + GAT(h)）防止深层训练的梯度消失，输入跳跃连接（cat([h_final, x_init])）将原始信道特征直送输出层，避免多层聚合后直传增益信息被稀释。结合LayerNorm稳定各层输入分布，有效缓解了深层图神经网络的过度平滑问题。

3. **多目标损失函数与无监督训练策略**：以倒数加权和速率为主目标（1/R），配合公平性（Jain指数）、QoS满足（L2平方惩罚）、SINR保底和功率约束共五项正则项，实现频谱效率与通信可靠性的多目标联合优化。训练阶段采用在线拓扑随机采样策略，GATNet的训练完全独立于WMMSE，具有良好的模块化解耦特性。

4. **GATNet+WMMSE两阶段混合方案**：这是本章的核心方法贡献。第一阶段利用训练好的GATNet进行快速前向推理（约5 ms）生成高质量初始功率分配；第二阶段以该初始解启动WMMSE迭代精调（仅8次迭代，约3 ms），在总计算延迟约8.31 ms的条件下达到WMMSE全迭代（60次）性能的97.2%。该混合方案实现了数据驱动方法（速度快、可泛化）与模型驱动方法（精度高、有理论保证）的优势互补，为GNN在无线资源管理中的实际部署提供了兼顾性能与实时性的可行方案。
