## 三、你的25篇文献中使用最多的功率分配方法

### 🥇 第一位：图神经网络方法（GNN-based Power Allocation）

**出现次数：13篇**（超过一半的核心论文）

这是2020年后最主流的方向。具体又可细分：

| 子类别 | 代表文献 | 特点 |
|---|---|---|
| **标准GNN/GCN** | [1] Eisen, [2] Shen, [3] Shen, [5] Zhang, [6] Guo, [10] Abode | 用消息传递聚合邻居干扰信息 |
| **图注意力网络(GAT)** | [7] Veličković(原论文), [13] JK-GAT, [14] Edge-GAT, [15] RIS+GAT | 可学习注意力权重，区分不同干扰源的重要性 |
| **算法展开(Unfolding)** | [4] Chowdhury, [21] Li | 将WMMSE迭代展开为GNN层，保留物理可解释性 |
| **联合资源分配** | [11] Marwani, [23] Chen | GNN同时处理功率+信道/频谱分配 |

**共同思路**：
```
信道状态矩阵 → 构建干扰图 → GNN消息传递 → 功率分配向量
```

### 🥈 第二位：WMMSE（加权最小均方误差）

**几乎所有13篇无线论文都将WMMSE作为性能上界基准**

WMMSE是功率分配问题的经典迭代优化算法：
- 通过引入辅助变量（MMSE接收机、MSE权重），将非凸和速率最大化问题转化为一系列凸子问题
- 迭代求解至KKT点，通常需要30~60次迭代
- 缺点：计算复杂度 O(N³·T_iter)，实时性差

### 🥉 第三位：算法展开（Algorithm Unfolding / Deep Unfolding）

**2篇核心论文**：[4] Chowdhury, [21] Li

这是近年兴起的方向，核心思想：
```
WMMSE迭代结构 → 展开为神经网络层 → 用数据学习各层参数
```
优势：既保留WMMSE的物理可解释性，又具备深度学习的推理速度。

### 第四位：强化学习方法

**0篇在你的列表**，但截图中的那篇论文使用了MARL。

MARL的思路是每个链路视为一个Agent，通过与环境交互学习最优功率策略。常见方法有Q-learning、DQN、Actor-Critic等。缺点：训练不稳定、状态空间爆炸。

## 四、趋势总结

```
2018年以前：WMMSE迭代优化为主，计算复杂度高
2018-2020：DNN/CNN端到端学习兴起，但丢失拓扑结构
2020-2022：GNN/GCN成为主流，利用图结构建模干扰关系
2022-2024：GAT取代GCN，注意力机制实现差异化干扰感知
2024-2026：GNN+Unfolding、联合资源分配、RIS/6G场景扩展
```

**你的GATNet定位**：属于2022-2024主线（GAT+边特征+无监督训练），同时融合了算法展开思想（GATNet+WMMSE混合方案）。你与主流方向的差异在于：
1. 边特征显式注入注意力（多数GAT论文只用节点特征）
2. 五合一损失函数（多数论文只用-sum_rate）
3. 跳跃连接保留直传增益（工程细节）

1. **传统方法**：WMMSE迭代优化 → 精度高但计算慢
